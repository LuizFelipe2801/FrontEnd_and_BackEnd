# Projeto Instagram - Testes Isolados

Este projeto é focado em criar funcionalidades pequenas e isoladas para demonstração no Instagram, usando front-end (HTML/CSS/JS, React) e back-end (Python, Node.js).

## Estrutura
- `frontend/`: Testes front-end isolados.
- `backend-python/`: Testes back-end em Python.
- `backend-nodejs/`: Testes back-end em Node.js.
- `docs/`: Documentação adicional.

## Como usar
Cada subpasta tem arquivos independentes para testes isolados. Rode individualmente e capture telas para postagens.
